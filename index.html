Lobber

Sharing files with BitTorrent

Download .zip Download .tar.gz View on GitHub

n. Lobber (clobber, clabber) a form of curdled milk. In Swedish lobber is called filbunke which in English translates as "bowl of files".
v. Lobber (lobbering) To send data using the Lobber service

This is Lobber, an application for sharing files using BitTorrent technology. There's a web interface as well as a RESTful http API.

Lobber was a project founded by SUNET. The Lobber project is now discontinued.

More Lobber repositories: Java Applet BitTorrent Client, Deluge plugin for Lobber, Standalone storage node for lobber
